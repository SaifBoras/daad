AMIGA files: 

EDI1.PRG -> English interpreter, loads PART1.DDB
EDI2.PRG -> English interpreter, loads PART1.DDB or PART2.DDB
SDI1.PRG -> Spanish interpreter, loads PART1.DDB
SDI2.PRG -> Spanish interpreter, loads PART1.DDB or PART2.DDB
PART1.CHR -> Charset
PART1.CHO -> Charset
PART1.DAT -> empty graphcis data
DAAD_Atari.ST - original Atari ST image that came with DAAD

Bundling releases:

use DRC to compile PART1.DDB with Atari ST target, add to disk
add interpreter, both charset, and graphics DB on disk

You can optionally put a PART1.SCR (Degas PI1) file on disk, will be shown then when the game is loaded.

for an example release check out Hibernated 1: http://8bitgames.itch.io
