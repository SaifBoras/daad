AMIGA files: 

edi1 -> English interpreter, loads PART1.DDB
edi2 -> English interpreter, loads PART1.DDB or PART2.DDB
sdi1 -> Spanish interpreter, loads PART1.DDB
sdi2 -> Spanish interpreter, loads PART1.DDB or PART2.DDB
PART1.CHR -> Charset
PART1.DAT -> empty graphcis data
S-Pic.adf - creates self-executing loading-screen image from IFF file
DAAD_Amiga.adf - original Amiga image that came with DAAD

Bundling releases:

use DRC to compile PART1.DDB with Amiga target, add to disk
add interpreter, charset, and graphics DB on disk

for an example release check out Hibernated 1: http://8bitgames.itch.io
